=================
Installation
=================

1. Create a Virtual Environment
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

::

   python3 -m venv venv

2. Activate the Virtual Environment
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Windows
^^^^^^^

::

   venv\Scripts\activate

macOS / Linux
^^^^^^^^^^^^^

.. code-block:: sh

   source venv/bin/activate

3. Install Module
~~~~~~~~~~~~~~~~~

::

   pip3 install .

4. Install Example Dependencies
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

::

   pip3 install -r examples/requirements.txt
